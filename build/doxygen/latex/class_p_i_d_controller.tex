A PID \doxylink{class_controller}{Controller} is a method of system control in which a correctional output is generated to guide the system toward a desired setpoint (aka target). The PID \doxylink{class_controller}{Controller} calculates the output based on the following factors:\+

Gains (proportional, integral, and derivative) Target Feedback

The gain values act as multipliers for their corresponding components of PID (more detail later). The target is the value which the system strives to reach by manipulating the output. The feedback is the system\textquotesingle{}s actual position or status in regards to the physical world. Another important term in PID is "{}error"{}, which refers to the difference between the target and the feedback.

Each of the three components of PID contributes a unique behavior to the system.

The Proportional component introduces a linear relationship between the error (target minus feedback) and the output. This means that as the feedback grows further away from the target, the output grows proportionally stronger. \begin{DoxyVerb}Proportional component = (P Gain) * (target - feedback)
\end{DoxyVerb}


The Integral component is designed to give a very precise approach of the feedback to the target. Depending on the scale of the physical system and the precision of feedback (e.\+g. sensors), the proportional component alone is likely not sufficient to provide adequate power (e.\+g. to motors) to guide the system in regards to small-\/scale corrections. The Integral component integrates the error of the system (target -\/ feedback) over time. If the system reaches a point where it is close but not exactly on top of the target, the integration will slowly build until it is powerful enough to overcome static resistances and move the system preciesly to the target. \begin{DoxyVerb}Integral component = (I Gain) * Integral of error over time

  *In this implementation, Integral is calculated with a running
  summation of the system's error, updated at each tick.
\end{DoxyVerb}


The Derivative component measures the rate of change of the feedback. It can reduce the strength of the output if the feedback is approaching the target too quickly or if the feedback is moving away from the target. \begin{DoxyVerb}Derivative component = (D Gain) * ((error - lastError) / time - lastTime)
\end{DoxyVerb}


The output generated by the PID \doxylink{class_controller}{Controller} is the sum of the three components.

PID output = Proportional component + Integral component + Derivative component Constructs the \doxylink{class_p_i_d_controller}{PIDController} object with PID Gains and function pointers for retrieving feedback (pid\+Source) and delivering output (pid\+Output). All PID gains should be positive, otherwise the system will violently diverge from the target.
