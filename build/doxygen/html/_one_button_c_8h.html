Detailed Description

Library for detecting button clicks, doubleclicks and long press pattern on a single button.

This is a port of the OneButton library to C to be used with the STM32 HAL framework.

CONCURRENCY NOTE: This library is designed for single-threaded usage. Do not call OB_Tick() from within callback functions. The library includes a reentrancy guard to prevent state corruption from recursive calls. If using in multi-threaded environment, external synchronization is required.
